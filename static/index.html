Frecipe Recipe API:

Frecipe is a free recipe API which serves recipes from a database and makes them accessible via HTTP APIs. Currently, it has more than 9500+ recipes, with more being added ocassionally. Web/Mobile apps can use the Frecipe API to get recipes easily.

*Note: API has a rate-limit of - 5 requests / minute
(while in the testing phase)

How does the Frecipe API work ?

If You want to use the Frecipe API in Your project, You can check out the Documentation or Alternative doc link

Example:

curl --header "Authorization: Token YOUR_TOKEN" {{ request.url }}api/recipes/?limit=2&skip=2 
const url = "{{ request.url }}api/recipes/?limit=2&skip=2"
const options = {
 "method": 'GET',
 "headers": {
 "Authorization": "Token YOUT_TOKEN"
 }
}
fetch(url, options)
 .then(response => response.json())
 .then(data => console.log(data))
 .catch(err => console.log(err.message))
  • cURL
  • JavaScript

As You have noticed from the example above, You need to have an API Token to be able to use the Frecipe API.
